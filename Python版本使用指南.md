## 📋 使用步骤

### 1. 基础使用（无需配置）
```bash
python momo_auto_reply.py
```

然后直接输入消息内容即可模拟收到聊天：
```
👤 您: 你好
🤖 AI回复: 你好啊！很高兴见到你！😊

👤 您: 今天天气怎么样？
🤖 AI回复: 天气挺不错的！记得多出去走走～
```

### 2. 高级配置（可选）
编辑 `momo_auto_reply.py` 文件，在 `AIReplyGenerator` 类中配置：

```python
# 配置您的API密钥
self.openai_api_key = "您的OpenAI密钥"
self.baidu_api_key = "您的百度文心密钥"

# 配置AI个性
self.character_setting = {
    "personality": "友善、幽默、聪明",
    "style": "自然对话风格",
    "response_length": "适中回复",
    "language": "中文"
}
```

### 3. 实用命令
在程序运行中输入：
- `help` - 查看帮助
- `status` - 查看运行状态  
- `config` - 查看当前配置
- `clear` - 清空对话历史
- `quit` - 退出程序
